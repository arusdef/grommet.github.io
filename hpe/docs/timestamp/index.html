Timestamp

A localized timestamp.

Options

align start|center|end
The horizontal alignment of the Timestamp.
fields date|time
Whether to show jsut the date or just the time.
value {ISO8601 string}|{Date}
The date value as an ISO8601 string or a Date object.

Examples

Default

<Timestamp value="2016-09-08T17:41:00.111Z" />

Date only

<Timestamp value="2016-09-08T17:41:00.111Z" fields="date" />

Time only

<Timestamp value="2016-09-08T17:41:00.111Z" fields="time" />
