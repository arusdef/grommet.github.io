Timestamp

A localized timestamp.

Usage

import Timestamp from 'grommet/components/Timestamp';
<Timestamp value={} />

Properties

align start|center|end
The horizontal alignment of the Timestamp.
fields date|time|year|month|day|hour|minute|second
Which fields to include in the Timestamp.
value {ISO8601 string}|{Date}
The date value as an ISO8601 string or a Date object.

Examples

Default

<Timestamp value="2016-11-19T16:30:37.501Z" />

Date only

<Timestamp value="2016-11-19T16:30:37.501Z" fields="date" />

Month and day only

<Timestamp value="2016-11-19T16:30:37.501Z" fields={["month", "day"]} />

Date and time with seconds

<Timestamp value="2016-11-19T16:30:37.501Z" fields={["date", "time", "seconds"]} />

Time only

<Timestamp value="2016-11-19T16:30:37.501Z" fields="time" />

Time with seconds

<Timestamp value="2016-11-19T16:30:37.501Z" fields={["time", "seconds"]} />

Hours only

<Timestamp value="2016-11-19T16:30:37.501Z" fields="hours" />
