Timestamp

A localized timestamp.

Usage

import Timestamp from 'grommet/components/Timestamp';
<Timestamp value={} />

Properties

align start|center|end
The horizontal alignment of the Timestamp.
fields date|time
Whether to show jsut the date or just the time.
value {ISO8601 string}|{Date}
The date value as an ISO8601 string or a Date object.

Examples

Default

<Timestamp value="2016-09-21T18:55:14.564Z" />

Date only

<Timestamp value="2016-09-21T18:55:14.564Z" fields="date" />

Time only

<Timestamp value="2016-09-21T18:55:14.564Z" fields="time" />
