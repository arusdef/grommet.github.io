NumberInput

A number input in a web form. We have a separate component from the browser base so we can style it.

Options

ariaDescribedby {string}
Optional attribute to enhance accessibility in case the this is used inside a context.
defaultValue {number}|{string}
Same as React <input defaultValue= >.
disabled true|false
Same as React <input disabled= >. Also adds a hidden input element with the same name so form submissions work.
id {string}
The DOM id attribute value to use for the underlying<input> element.
label {string|node}
Label text to place next to the control.
max {number}
Maximum value.
min {number}
Minimum value.
name {string}
The DOM name attribute value to use for the underlying<input> element.
onChange {function}
Same as React <input onChange= >.
step {number}
Steps to increase and decrease by.
toggle true|false
Whether to visualize it as a toggle switch.

Example

Basic

<Form>
  <FormField>
    <NumberInput id="item2" name="item2" label="Item 2" value={10} onChange={...} />
  </FormField>
</Form>

Disabled

<Form>
  <FormField>
    <NumberInput id="item4" name="item4" label="Item 4" disabled={true} value={10} />
  </FormField>
</Form>
