CheckBox

A check box in a web form. We have a separate component from the browser base so we can style it.

Options

checked true|false
Same as React <input checked= >.
defaultChecked true|false
Same as React <input defaultChecked= >.
disabled true|false
Same as React <input disabled= >. Also adds a hidden input element with the same name so form submissions work.
id {string}
The DOM id attribute value to use for the underlying<input> element.
label {string|node}
Label text to place next to the control.
name {string}
The DOM name attribute value to use for the underlying<input> element.
onChange {function}
Same as React <input onChange= >.
reverse true|false
Whether to show the label in front of the checkbox. Defaults to false.
toggle true|false
Whether to visualize it as a toggle switch. Defaults to false.

Examples

Basic

<CheckBox id="item1" name="item1" label="Item 1" checked={false} onChange={...} />

Toggle

<CheckBox id="item2" name="item2" label="Item 2" checked={false} toggle={true} onChange={...} />

Disabled

<CheckBox id="item3" name="item3" label="Item 3" checked={false} disabled={true} />

Disabled Toggle

<CheckBox id="item4" name="item4" label="Item 4" checked={false} toggle={true} disabled={true} />

Reverse

<CheckBox id="item5" name="item5" label="Item 5" checked={false} reverse={true} onChange={...} />

No Label

<CheckBox id="item6" name="item6" checked={false} onChange={...} />
