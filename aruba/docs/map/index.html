Map

Shows a graph of nodes and links between them.

  1. First category
    • First item
    • Second item
    • Third item
  2. Second category
    • Fourth item
    • Fifth item
  3. Third category
    • Sixth item

Usage

import GrommetMap from 'grommet/components/Map';
<GrommetMap data={...} />

Properties

data { categories: [{id: , label: , items: [id: , node: ]}, ...], links: [{parentId: , childId: }, ...] }
An array of objects describing the data.
vertical true|false
Whether to orient the map vertically.
