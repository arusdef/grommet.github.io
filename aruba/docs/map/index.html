Map

Shows a graph of nodes and links between them.

Options

data { categories: [{id: , label: , items: [id: , node: ]}, ...], links: [{parentId: , childId: }, ...] }
An array of objects describing the data.
vertical true|false
Whether to orient the map vertically.

Example

  1. First category
    • First item
    • Second item
    • Third item
  2. Second category
    • Fourth item
    • Fifth item
  3. Third category
    • Sixth item
    • Seventh item

  "categories": [
    {

      "items": [
        {"id": "item-1-1", "node": "First item"},
        {"id": "item-1-2", "node": "Second item"},
        {"id": "item-1-3", "node": "Third item"}
      ]
    },
    {

      ]
