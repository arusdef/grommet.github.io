Main Content

Timestamp

Examples

A localized timestamp.

Properties

align start|center|end
The horizontal alignment of the Timestamp.
fields date|time|year|month|month-full|day|hour|minute|second
Which fields to include in the Timestamp.
value {ISO8601 string}|{Date}
The date value as an ISO8601 string or a Date object.

Examples

Default

<Timestamp value='2017-03-16T21:03:19.720Z' />

Date only

<Timestamp value='2017-03-16T21:03:19.720Z'
  fields='date' />

Month and day only

<Timestamp value='2017-03-16T21:03:19.720Z'
  fields={['month', 'day']} />

Date and time with seconds

<Timestamp value='2017-03-16T21:03:19.720Z'
  fields={['date', 'time', 'seconds']} />

Time only

<Timestamp value='2017-03-16T21:03:19.720Z'
  fields='time' />

Time with seconds

<Timestamp value='2017-03-16T21:03:19.720Z'
  fields={['time', 'seconds']} />

Hours only

<Timestamp value='2017-03-16T21:03:19.720Z'
  fields='hours' />
